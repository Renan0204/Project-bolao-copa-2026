- Criar perfis USER e ADMIN

## Etapa 3 - Seleções

- Criar entidade Selecao
- Criar CRUD de seleções para admin

## Etapa 4 - Partidas

## Etapa 5 - Palpites

- Criar entidade Palpite

## Etapa 6 - Resultados e Pontuação

- Lançar resultado da partida

- Listar ranking geral
